Бот для расписания и домашки

Telegram-бот: показывает расписание на выбранный день и домашнее задание по предметам.

Возможности
Просмотр расписания по дням
Просмотр домашнего задания по предметам
Админ-панель: добавление/удаление предметов, редактирование ДЗ

#Запуск
1. Установи зависимости: `pip install python-telegram-bot python-dotenv`
2. Создай файл `.env` с токеном: `BOT_TOKEN=твой_токен`
3. Запусти: `python schedule.py`

Админ
Админ задаётся через переменную окружения `ADMIN_ID`.
